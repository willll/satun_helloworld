# satun_helloworld
SEGA Saturn HelloWorld with SBL

Based on smpdbg10 SBL sample code.

- How to build :

rm -rf * && cmake -DCMAKE_INSTALL_PREFIX=/saturn .. && make && make install

(see https://github.com/willll/saturn-docker for a toolchain)
